Daily Flow - Aplicación de Organización Personal
Aplicación móvil híbrida desarrollada con Ionic Angular para la gestión y organización de tareas diarias en diferentes categorías.

✅ Sistema de Autenticación - Login y registro de usuarios
✅ Seguridad con AuthGuard - Rutas protegidas para usuarios autenticados
✅ Persistencia con SQLite - Base de datos local para usuarios y datos
✅ Consumo de API REST - Integración con JSONPlaceholder
✅ Modo Offline - Persistencia de datos cuando no hay conexión
✅ Plugin de Cámara - Captura de fotos usando Capacitor
✅ Plugin de Geolocalización - Visualización de ubicación en mapas
✅ Navegación Optimizada - Rutas lazy-loading para mejor rendimiento

Tecnologías Utilizadas

Ionic Framework 7+
Angular 17+
Capacitor 5+
SQLite - Base de datos local
TypeScript
RxJS - Manejo de observables
